СИСТЕМА АРХИВИРОВАНИЯ И ПЕРЕДАЧИ ИЗОБРАЖЕНИЙ (PACS)

AP/IX - Программно-аппаратный комплекс для автоматизации рабочего процесса получения, хранения, обработки, просмотра, маршрутизации диагностических медицинских изображений и данных.

Основные Функции AP/IX:

Прием, хранение, передача, поиск исследований, уникальная идентификация, автоматическое и ручное разрешение конфликтов, обеспечение информационной безопасности хранимых данных, обеспечение аудита действий пользователей

Получение

  • Обеспечивает прием и отображение изображений, поступивших с высокоспециализированных диагностических устройств и цифровую обработку перед предоставлением пользователям для анализа.
  • Поддержка приема, хранения, визуализации, передачи радиологических исследований в соответствии с требованиями стандарта DICOM 3.0

Хранение

  • Поддержка операционных классов DICOM (SOP Classes):
  • Поддержка хранения и отображения объектов: DICOM SR, DICOM KOS, DICOM GSPS,

DICOM Encapsulated PDF

  • Долгосрочное решение для архивирования, резервного копирования, восстановления данных исследований.
  • Многоуровневое хранение и правила маршрутизации исследований, основанные на ваших требованиях.
  • Поддержка и управление жизненным циклом изображений
  • Сокращение объем ручных задач по архивированию исследований и поддержка оптимального использования хранилища.

Передача Обеспечивает передачу исcледований по протоколу DICOM c диагностических устройств на сервер системы PACS, а также передачу исследований из стронних DICOM – архивов

DICOM Encapsulated PDF

  • Работа с рабочими списками исследований
  • Работа с релевантными (предыдущими) исследованиями
  • Открытие изображений определенного исследования
  • Расширенный поиск исследований
  • Работа с внешними носителями (экспорт и импорт).
  • Мониторинг статуса задания
